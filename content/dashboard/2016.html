Please note that software release 20.0 of the Network Manager on 04 April 2016 introduced a change in the computation of ATFM delay for operational purposes. For consistency reasons, ATFM delays on this web page will continue to be based on the methodology used before April 2016. As a result, the data published on this website might deviate from the data directly published by EUROCONTROL for operational purposes. See the Meta data for more information on the changes in NM release 20.0.
